Por que Markdown?

Markdown uma linguagem de marcao leve que permite formatar texto de forma simples e eficiente.

Vantagens:

  • Sintaxe simples e intuitiva
  • Compatvel com qualquer editor de texto
  • Facilmente convertvel para HTML
  • Amplamente suportado em plataformas
