Por que Markdown?

Markdown é uma linguagem de marcação leve que permite formatar texto de forma simples e eficiente.

Vantagens:

  • • Sintaxe simples e intuitiva
  • • Compatível com qualquer editor de texto
  • • Facilmente convertível para HTML
  • • Amplamente suportado em plataformas
